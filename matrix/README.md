MegaMatrix
==========

Exercise:

Please implement in Java a method which prints matrix of special structure.
Signature of method: 
```java
void printMegaMatrix(int n)
```
This method should print to the screen matrix with the size [2n-1] x [2n-1].

Examples:

```
n = 2
*1*
121
*1*

n = 3
**1**
*121*
12321
*121*
**1**

n = 5
****1****
***121***
**12321**
*1234321*
123454321
*1234321*
**12321**
***121***
****1****
```
