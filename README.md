Playground
==========

Just another skill test.

Original exercise:

Please write code of java method which print matrix.

Signature of method: `void printMegaMatrix(int n)`.
Method should print to screen matrix with the size [2n-1] x [2n-1].

Examples:

```
n = 2

*1*
121
*1*

n = 3

**1**
*121*
12321
*121*
**1**

n = 5

****1****
***121***
**12321**
*1234321*
123454321
*1234321*
**12321**
***121***
****1****
```
